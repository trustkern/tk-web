# TrustKern web (marketing)

Static single-page site: dark industrial aesthetic, gateway-blind / E2E positioning.

- **Entry:** `index.html` (self-contained — all CSS, JS, and favicon inline)
- **External dependencies:** Google Fonts (DM Sans, Outfit) loaded via CDN; `assets/apple-touch-icon.png` for iOS bookmarks; `assets/logo-512.png` for OG image.
- **Sections:** Hero, value strip, problem, solution, how-it-works, architecture, why, features, industries, security, deployment, contact, CTA, footer.

**Preview:** `python3 -m http.server 8080` from this directory.

Design notes: **`DESIGN.md`**.
